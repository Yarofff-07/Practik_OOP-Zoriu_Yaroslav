# Практична робота №1
## Тема: Робота з аргументами командного рядка в Java

---

## Мета роботи

Ознайомитись із принципом передачі аргументів командного рядка у програму  
та реалізувати просту консольну програму мовою **Java**, яка отримує та
виводить передані аргументи.

---

## Постановка задачі

1. Підготувати репозиторій для розміщення проекту.
2. Створити консольну програму на Java, яка приймає аргументи
командного рядка.
3. Вивести отримані аргументи у консоль разом з їх кількістю.
4. Додати до репозиторію посилання на GitHub та приклад виконання програми.

---

## Реалізація

У програмі реалізовано просту обробку аргументів командного рядка.

Основні можливості програми:

- отримання аргументів при запуску програми
- визначення кількості аргументів
- виведення кожного аргументу окремим рядком
- перевірка ситуації, коли аргументи відсутні

Якщо аргументи не передані, програма повідомляє користувача та
показує приклад правильного запуску.

---

## Prac1.java

`src/Prac1.java` — головний файл програми.  
Містить метод `main()`, який приймає аргументи командного рядка,
перевіряє їх наявність та виводить у консоль.

<details>
<summary>Код</summary>

```java
public class Prac1 {
    public static void main(String[] args) {

        System.out.println("=== Вивід аргументів командного рядка ===");

        if (args.length == 0) {
            System.out.println("Аргументи не передані.");
            System.out.println("Приклад запуску:");
            System.out.println("java Prac1 hello world 123");
            return;
        }

        System.out.println("Кількість аргументів: " + args.length);
        System.out.println("Список аргументів:");

        for (int i = 0; i < args.length; i++) {
            System.out.println((i + 1) + ") " + args[i]);
        }
    }
}
```
</details>

## Компіляція програми

```
javac Prac1.java
```

## Запуск програми

```
java Prac1 argument1 argu2 arg3 4
```
